# Odoo 17 ERP — Package XL: Manufacturing

Full-scale ERP package with MRP capabilities, built on top of Package L (Service Business).

## Key Features

- **MRP (Manufacturing)**:
  - Manufacturing Orders (MO) with BOM (Bill of Materials)
  - Track production status and MO stages
  - Manage product BOM structure
  - Work Orders for each production step aligned with BOM
  - Work Centers for resource and machine tracking
  - Routing to define production sequences and workstations

- **Perfect ERP (All-in-One)**:
  - Full service business features
  - Full retail/distributor features
  - Full e-commerce features
  - Advanced inventory management
  - Complete accounting integration
  - Executive dashboard
  - Mobile approval
  - Document workflows
  - Full POS integration

## Includes all features from Package S, M & L:
- Sales, Purchasing, RFQ, Inventory, Accounting, POS, Dashboard, Workflows, Mobile App

## Tech Stack

- Python 3.10+
- Odoo 17 Community/Enterprise
- PostgreSQL
- XML (Views, Reports, Data)
